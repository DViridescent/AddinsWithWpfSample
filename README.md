## 使用方式
Clone仓库到本地
使用最新的Visual Studio打开项目

- 情况1：单独运行WPF项目->运行Sample.UI.WPF.Entry
- 情况2：在某个软件（AutoCAD、Rhino、Revit）中运行->进入对应文件夹，阅读对应的README.md文件
